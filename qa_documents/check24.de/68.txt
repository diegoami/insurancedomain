Wie funktioniert der CHECK24-Vergleichsrechner?

Mit dem CHECK24-Vergleichsrechner finden Sie schnell und einfach eine günstige Hundehaftpflichtversicherung. Sämtliche Daten werden streng vertraulich behandelt und mit einer sicheren Verschlüsselungstechnik übertragen. Somit haben Dritte keinen Zugriff auf Ihre persönlichen Daten. Um Sie mit der Bedienung des Vergleichsrechners vertraut zu machen, haben wir eine Kurzanleitung erstellt.

Schritt 1: Anzahl der Hunde eingeben
Schritt 2: Leistungs- und Vertragsdetails wählen
Schritt 3: Übersichtlicher Leistungs- und Tarifvergleich
 




Anzahl der Hunde eingeben

Button „vergleichen“ anklicken

 



Nachdem Sie im ersten Schritt die Anzahl der Hunde eingegeben haben, wird Ihnen bereits eine Vielzahl von Hundehaftpflichtversicherungen angezeigt. Nun können Sie weitere Angaben machen, um die Vertragsbedingungen und den Leistungsumfang individuell anzupassen.

Anzahl der Hunde: Hier können Sie die Anzahl der zu versichernden Hunde verändern.

Anzahl der Kampfhunde: Wenn Sie einen Hund versichern möchten, der von den Behörden aufgrund seiner Rasse als Listenhund geführt wird und umgangssprachlich meist als „Kampfhund“ bezeichnet wird, nutzen Sie bitte diese Eingabemöglichkeit.

Zahlweise: Hier können Sie zwischen einer jährlichen, halb- oder vierteljährlichen Zahlweise auswählen. Den Beitrag jährlich zu bezahlen, ist meistens am günstigsten, weil für die Versicherer der Verwaltungsaufwand geringer ist.

Selbstbeteiligung: Wenn Sie möchten, können Sie eine generelle Selbstbeteiligung vereinbaren – das senkt den Beitrag.

Deckungssumme: Hier können Sie bestimmen, welche Deckungssumme die Tarife mindestens haben sollen. Unsere Experten empfehlen eine Summe von mindestens fünf Millionen Euro.

Mindestlaufzeit: Wenn Sie eine Vertragslaufzeit von drei Jahren wählen, gewähren einige Anbieter einen Beitragsrabatt. Damit sind Sie allerdings für drei Jahre an den Versicherer gebunden. Somit können Sie frühestens nach drei Jahren den Versicherer wechseln.

Öffentlicher Dienst: Manche Versicherer bieten Sondertarife für Mitarbeiter des öffentlichen Dienstes an.

Wenn Sie einen Faktor verändern, passt sich die Darstellung an Ihre Eingaben an. Die Sortierung Ihrer Tarifangebote können Sie mithilfe folgender Reiter ändern:

Beitrag: Wenn Sie den Reiter „Beitrag“ per Mausklick aktivieren, werden die angebotenen Tarife nach dem Beitrag sortiert. Anfangs sind die Tarife immer nach dem günstigsten Tarif angeordnet. Diese Sortierung können Sie über den Reiter „Beitrag“ umkehren.

Versicherung: Wenn Sie den Sortiermodus „Versicherung“ wählen, werden Ihnen die verfügbaren Versicherer in alphabetischer Reihenfolge angezeigt.

Note: Unsere CHECK24-Versicherungsexperten haben sich die Tarife der Versicherer genau angeschaut und die Leistungen nach bestimmten Kriterien bewertet. Wenn Sie „Note“ anklicken, richtet sich die Rangfolge nach dieser Bewertung.

Tipp: Sehr hilfreich bei der Wahl der Versicherung sind auch die Kundenbewertungen. Diese können Sie sehen, wenn Sie unter der CHECK24-Tarifnote auf „Bewertungen“ klicken.

Nachdem Sie sich einen Überblick verschafft haben, können Sie über den Button „Online-Antrag“ eine Versicherung über eine sichere Datenverbindung online abschließen oder sich ein Angebot per E-Mail und Post zusenden lassen. Wenn Sie allerdings noch mehr Informationen benötigen, können Sie mithilfe der Funktion „Details anzeigen“ die Leistungen mehrerer Tarife miteinander vergleichen. Im dritten Schritt erklären wir Ihnen, wie Sie einen detaillierten Tarifvergleich vornehmen.

Wenn Sie bei einem Tarif „Details anzeigen“ anklicken, erhalten Sie folgende Ansicht.



Beim Detail-Tarifvergleich sehen Sie die wichtigsten Leistungsmerkmale des gewählten Tarifs. Damit können Sie auf einen Blick erkennen, ob und bis zu welchen Höchstsummen der Tarif die gewünschten Leistungen enthält. Wenn Sie den gewählten Tarif mit einem anderen Tarif vergleichen möchten, klicken Sie einfach auf „weiteren Tarif hinzufügen“. Sie können bis zu drei Tarife gleichzeitig miteinander vergleichen.

Möchten Sie die Anzahl der angezeigten Leistungsmerkmale reduzieren, können Sie von „Alle Leistungen“ auf „Wichtigste Leistungen“ umstellen. Schneller, einfacher und übersichtlicher ist ein Tarifvergleich kaum möglich. Wenn Sie eine Eingabe ändern möchten, brauchen Sie nur auf „zurück zur Ergebnisliste“ klicken.
